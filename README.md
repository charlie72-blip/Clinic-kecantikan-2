# Klinik Aesthetic — Vite + React + Tailwind

This project is a starter for a premium aesthetic clinic landing page built with Vite, React, Tailwind CSS and Framer Motion.

Quick start

1. Install dependencies

```bash
npm install
```

2. Start dev server

```bash
npm run dev
```

Notes

- Update the WhatsApp placeholder number `62XXXXXXXXXX` in `src/components/Booking.jsx`, `src/components/Location.jsx`, and `src/components/Footer.jsx`.
- Tailwind is configured in `tailwind.config.cjs` and styles live in `src/index.css`.
